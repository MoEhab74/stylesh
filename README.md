## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Flutter SDK
- Dart SDK
- Android Studio / VS Code
- Emulator or physical device

---

### Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/stylesh.git
```

2. Navigate to the project folder

```bash
cd stylesh
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the app

```bash
flutter run
```
